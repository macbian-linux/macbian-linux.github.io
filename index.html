Macbian Linux

This is the homepage for the currently very unfinished project focused on providing an easy and efficient installer for Debian Linux for Intel Macs, from the original 2006 models to the last pre-T2 Macs released. Macbian will use a customized KDE Plasma or MATE desktop environment by default, although a command-line only release will also be provided.

Key Features

- Installation through a single terminal command

- No non-standard dependencies required on macOS (e.g. Python)

- Automatic installation of Broadcom WiFi driver packages (b43, broadcom-sta)

- Automatic patcher that applies machine-specific fixes (e.g. NVIDIA graphics, faulty AMD graphics disabling on 2011 MacBooks, sound fixes)

- Prebuilt ISO images for flashing with e.g. balenaEtcher

- Built off latest Debian Stable builds.

Current Status

Check out the GitHub project page for the current status of the Macbian installer. As of October 4, 2022, the Macbian installer is not functional, but I expect a working prototype to be ready in a few weeks.
